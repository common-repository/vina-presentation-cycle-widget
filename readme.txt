A great WordPress Slider to display featured posts in Presentation Cycle.

== Description ==

The Vina Presentation Cycle Widget for WordPress is a good choice for showing serial content to bring more efficiency transferring information to readers. Instead of generating a list of numbers that are clickable Presentation Cycle generates a progress bar that shows when the new slide will appear.

Features:

- Support multi Wordpress Version.
- Get title, description from post item.
- Load image from Feature Images.
- Load post items from a specific category.
- Allows to adjust the number, ordering of post item displayed.
- Customizable speed, delay, size etc.
- Cycle through elements
- Adjustable animation times
- Generates a navigation bar that also shows the progress
- Works in modern browsers
- Allows to show/hide title, image, description and readmore button.
- Allows resize image before display.
- Languages included: English
- Allow loading wordpress widget inside post, page.
- To manipulate almost everything !
- Totally customizable and skinnable using HTML and CSS
- Lightweight, modern and fast-loading design.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Tableless design and 100% CSS-based
- Fully compatible IE8+, Firefox 10+, Flock 0.7+, Netscape, Safari, Opera 9.5
- Easy installation

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire 'vina-presentation-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'Vina Presentation Cycle Widget' to your sidebar
